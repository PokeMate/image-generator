# image-generator

## Installation

### Without Docker

Create a virtualenv and activate it:

```bash
python3 -m venv venv
. venv/bin/activate
```

Or windows:

```bash
py -3 -m venv venv
venv\Scripts\activate.bat
```

Install all the dependencies.

```bash
pip3 install -r requirements.txt
```

Start the API

```bash
python3 api/main.py
```
